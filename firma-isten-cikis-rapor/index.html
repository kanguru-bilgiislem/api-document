SGK İşten Çıkış Raporu

Url Metot
firma-isten-cikis-rapor GET

Açıklama

Bu endpoint'i kullanarak SGK'ya yapılan işten çıkış işlemlerini takip edebilirsiniz.

Request

Query String Parametreleri

Parametre Açıklama Zorunlu Mu?
secretKey Size verilen firmanıza ait secretKey ✔️
startDate Başlangıç tarihi. YYYY-MM-DD formatında olmalıdır. ✔️
finishDate Bitiş tarihi. YYYY-MM-DD formatında olmalıdır. ✔️
tarihTip S,E veya K değeri alabilir. Bu değere göre tarih filtreleri farklı çalışacaktır. S: (İşten Çıkış Tarihi) E: İşlem (Evrak) Tarihi K: Kanguru sistemine düşme tarihi ✔️

Örnek Çağrı https:/api.teknokanguru.com.tr/externalapi/firma-isten-cikis-rapor?secretKey=XXXXXXXX&startDate=2025-09-01&finishDate=2026-09-17&tarihTip=S

Response

Örnek

  "isSuccess": true,

  "items": [
     {
      "sicil": "ABC",

Header

Ad Tip Açıklama
isSuccess boolean İşlemin başarılı olup olmadığını belirtir
errorMessage string Genel hata mesajı (varsa)
items Item[] Detay sonuçların listesi

Item:

Ad Tip Açıklama
sicil string Personelin çıkış yaptığı işyerinin Kanguru sistemindeki adı
sicilNo string Personelin çıkış yaptığı işyerinin SGK sicil nosu
grup string Personelin çıkış yaptığı işyerinin Kanguru sistemindeki grubu
adSoyad string Personelin adı ve soyadı
tcKimlikNo string Türkiye Cumhuriyeti kimlik numarası (11 haneli)
meslekKod string Meslek kodu (örn. 5242.10)
meslekAd string Meslek adı (örn. Satış Danışmanı / Uzmanı Yardımcısı)
referansNo string İşten çıkış işleminin SGK tarafından verilen referans numarası
istenCikisTarihi datetime İşten çıkış tarihi
istenCikisKodu integer İşten çıkış nedeni kodu
islemTarihi datetime SGK tarafında çıkış işleminin yapıldığı tarih (Evrak Tarihi)
createDate datetime Çıkışın Kanguru sistemine düşme tarihi
