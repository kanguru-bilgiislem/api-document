SGK İşe Giriş Raporu

Url Metot
firma-ise-giris-rapor GET

Açıklama

Bu endpoint'i kullanarak SGK'ya yapılan işe giriş işlemlerini takip edebilirsiniz.

Request

Query String Parametreleri

Parametre Açıklama Zorunlu Mu?
secretKey Size verilen firmanıza ait secretKey ✔️
startDate Başlangıç tarihi. YYYY-MM-DD formatında olmalıdır. ✔️
finishDate Bitiş tarihi. YYYY-MM-DD formatında olmalıdır. ✔️
tarihTip S,E veya K değeri alabilir. Bu değere göre tarih filtreleri farklı çalışacaktır. S: (İşe Gİriş Tarihi) E: İşlem (Evrak) Tarihi K: Kanguru sistemine düşme tarihi ✔️

Örnek Çağrı https:/api.teknokanguru.com.tr/externalapi/firma-ise-giris-rapor?secretKey=XXXXXXXX&startDate=2025-09-01&finishDate=2026-09-17&tarihTip=S

Header

Ad Tip Açıklama
isSuccess boolean İşlemin başarılı olup olmadığını belirtir
errorMessage string Genel hata mesajı (varsa)
items Item[] Detay sonuçların listesi

Item:

Ad Tip Açıklama
sicil string Personelin giriş yaptığı işyerinin Kanguru sistemindeki adı
sicilNo string Personelin giriş yaptığı işyerinin SGK sicil nosu
grup string Personelin giriş yaptığı işyerinin Kanguru sistemindeki grubu
adSoyad string Personelin tam adı ve soyadı
ad string Personelin adı
soyad string Personelin soyadı
tcKimlikNo string Türkiye Cumhuriyeti kimlik numarası (11 haneli)
referansNo string İşe giriş işleminin SGK tarafından verilen referans numarası
iseGirisTarihi datetime İşe giriş tarihi
islemTarihi datetime SGK tarafında giriş işleminin yapıldığı tarih (Evrak Tarihi)
createDate datetime Girişin Kanguru sistemine düşme tarihi
meslekKod string Meslek kodu (örn. 1213.23)
meslekAd string Meslek adı (örn. Müdür Yardımcısı)
anneAdi string Personelin anne adı
babaAdi string Personelin baba adı
dogumYeri string Personelin doğum yeri
dogumTarihi datetime Personelin doğum tarihi
isEngelli boolean Personelin engelli olup olmadığı
sigortaKodu string Personelin Sigorta kodu
isKismi boolean Personelin kısmi süreli çalışıp çalışmadığı
ogrenimDurumu string Personelin Öğrenim durumu
sgkSicilNo string Personelin SGK sicil numarası
