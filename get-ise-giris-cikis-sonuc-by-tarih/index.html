İşe Giriş/Çıkış Sonuçları (Tarih ile)

Url Metot
get-ise-giris-cikis-sonuc-by-tarih GET

Açıklama

Bu endpoint'i kullanarak bir bildirim içindeki SGK'ya gönderilen işe giriş/çıkışların sonuçlarını görebilirsiniz.

Request

Query String Parametreleri

Parametre Açıklama Zorunlu Mu?
secretKey Size verilen firmanıza ait secretKey ✔️
startDate Başlangıç tarihi. YYYY-MM-DD formatında olmalıdır. ✔️
finishDate Bitiş tarihi. YYYY-MM-DD formatında olmalıdır. ✔️

Örnek Çağrı https://api.teknokanguru.com.tr/externalapi/get-ise-giris-cikis-sonuc-by-tarih?secretkey=abc123&startDate=2024-12-01&finishDate=2024-12-31

Response

Örnek

{
  "isSuccess": true,
  "errorMessage": null,
  "items": [
    {
      "tcKimlikNo": "12345678901",
      "statusId": 104,
      "errorMessage": null,
      "referansNo": "REF123456",
      "bildirimId": 111,
      "isGiris": true
    },
    {
      "tcKimlikNo": "10987654321",
      "statusId": 105,
      "errorMessage": "lorem ipsum",
      "referansNo": null,
      "bildirimId": 1234,
      "isGiris": false
    }
  ]
}

Header

Ad Tip Açıklama
isSuccess boolean İşlemin başarılı olup olmadığını belirtir
errorMessage string Genel hata mesajı (varsa)
items Item[] Detay sonuçların listesi
Item:
Ad Tip Açıklama
tcKimlikNo string Kişinin T.C. Kimlik numarası
statusId integer İşlem sonucu durum kodu (Detayı aşağıda)
errorMessage string Kişiye özel hata mesajı (varsa)
referansNo string Bildirime ait SGK'dan verilen referans numarası
bildirimId integer Sistem tarafındaki bildirim ID’si
isGiris boolean true: Giriş bildirimi, false: Çıkış bildirimi
kayitTarihi date Bildirimin kayıt tarihi

Tanımlamalar

Status Değerleri

StatusId Açıklama
102 Henüz SGK'ya bildirilmedi
103 Veri tutarlığı kontrolünde Hata Aldı
104 SGK'ya başarıyla bildirildi
105 SGK sisteminde hata aldı
